# ⚡️ TeamSpirit 打刻お助けセット for Hammerspoon

打刻を忘れないようにしてくれる、2つの小さなプログラムのセットです。

| プログラム | いつ動く? | なにをする? |
|---|---|---|
| 🔔 `teamSpiritReminder.lua` | パソコンを起動 / スリープ解除したとき | 今日まだ TeamSpirit を開いていなければ、大きなお知らせを出す |
| ⚡️ `powerMenu.lua` | `⌥`(option) + `X` を押したとき | きれいな電源メニューを出して、シャットダウンのときだけ「退勤打刻した?」と聞いてくれる |

どちらも **Hammerspoon** という無料アプリの中で動きます。

⚠️ **これは Mac 専用です。** Windows では動きません。

---

## 1. これは何をするもの?

### 🔔 出勤リマインダー(teamSpiritReminder.lua)

- Mac を起動 / スリープ解除したときに、こっそりチェックします
- 「今日はまだ TeamSpirit を開いていないな」と気づいたら、
  画面いっぱいのお知らせを出します
- お知らせのボタンを押すと、TeamSpirit のページが開きます
- 1回開いたら、その日はもう出てきません(次の日また出ます)

### ⚡️ 電源メニュー(powerMenu.lua)

`⌥` + `X` を押すと、画面のまん中にこんなメニューが出てきます。

```
      Power Menu

  Restart          R
  Sleep            S
  Shutdown         U
  Lock             L

↑↓ Select   Enter Execute   Esc / ⌥X Cancel
```

`↑` `↓` でえらんで `Enter`。または `R` `S` `U` `L` のキーを直接おす。
それだけで、パソコンをそうさできます。

`Shutdown` をえらんだときだけ、もうひとつ画面が出てきます。

```
      退勤打刻

  TeamSpiritで
  退勤打刻しましたか?

  Open TeamSpirit  T
  Continue         C

↑↓ Select   Enter Execute   Esc Back
```

`T` を押すと TeamSpirit が開きます。打刻をわすれずにできる、というわけです。
打刻がおわったら `C`(または `↓` でえらんで `Enter`)を押すと、シャットダウンされます。

---

## 2. 必要なもの

- Mac 本体(macOS が入っているパソコン)
- インターネット接続
- 「Hammerspoon(ハマースプーン)」という無料アプリ
  - パソコンを自動で操作するための道具箱のようなアプリです
  - この2つのプログラムは、どちらも Hammerspoon の中で動きます

---

## 3. Hammerspoon をインストールする

1. ブラウザ(SafariやChromeなど)で下のURLを開きます

   ```
   https://www.hammerspoon.org/
   ```

2. 画面の中にある **「Download」** と書かれたボタンをクリックします
3. ダウンロードした `Hammerspoon-x.x.x.zip` のようなファイルを
   ダブルクリックして展開(解凍)します
4. 出てきた **「Hammerspoon.app」** を、
   **「アプリケーション」フォルダ(Applications フォルダ)** の中に
   ドラッグ&ドロップでコピーします

   > 💡 「アプリケーション」フォルダは、Finder の左側にある
   > 「アプリケーション」というアイコンをクリックすると出てきます。

5. コピーした **Hammerspoon.app** をダブルクリックして起動します
6. メニューバー(画面の一番上の帯)に、Hammerspoon の小さいアイコン
   (山のマーク🏔️)が出てきたら成功です

### 3-1. 最初だけ必要な「許可」の設定

Hammerspoon は、画面いっぱいにお知らせを出したり、キー操作を受け取ったりするために
「アクセシビリティ」という特別な許可が必要です。

1. 「システム設定」を開きます
2. 「プライバシーとセキュリティ」→「アクセシビリティ」を選びます
3. 一覧の中から **Hammerspoon** を探して、チェックを ON にします

   > 💡 一覧に出てこない場合は、いったん Hammerspoon を終了して
   > もう一度起動すると出てくることが多いです。

---

## 4. ファイルを置く

1. Finder を開きます
2. メニューの **「移動」→「フォルダへ移動...」** をクリックします

   > 💡 Windowsでいう「アドレスバーに直接パスを入力する」ような機能です。

3. 出てきた入力欄に、次のように入力して Enter を押します

   ```
   ~/.hammerspoon
   ```

   - `~` は「自分のホームフォルダ」という意味の記号です
   - もし `.hammerspoon` フォルダが無いと言われた場合は、
     Finder で新規フォルダを作り、名前を `.hammerspoon` に
     してから、もう一度この手順をやってみてください
     (フォルダ名は先頭のピリオド `.` を忘れずに)

4. 配布された次のファイルを、すべてこの `~/.hammerspoon` フォルダの中に
   そのままコピーします。

   - `init.lua` ← Hammerspoon が最初に読み込む設定ファイル
     (すでに `init.lua` がある場合は、中身を確認して下の 4-1 を追記してください)
   - `powerMenu.lua` ← 電源メニュー本体
   - `teamSpiritReminder.lua` ← 出勤リマインダー本体

   `teamspirit_last_open.txt` というファイルは、
   プログラムが自動で作るファイルなので、自分でコピーする必要はありません。

### 4-1. `init.lua` の中身

`init.lua` の中に、次の2行が入っていることを確認してください。
入っていなければ、この2行を追加して保存します。

```lua
require("powerMenu")
require("teamSpiritReminder")
```

> 💡 これは「`powerMenu.lua` と `teamSpiritReminder.lua` を読み込んでね」
> という意味の命令です。2つとも書かないと、片方しか動きません。

---

## 5. 設定を反映させる(リロード)

1. メニューバーの Hammerspoon アイコン(🏔️)をクリックします
2. **「Reload Config」** をクリックします

   > 💡 「設定ファイルを読み直してね」というボタンです。
   > ファイルを書き換えたときは、必ずこれを押します。

3. もし今日まだ TeamSpirit を開いていなければ、
   画面いっぱいにお知らせが表示されるはずです 🎉
4. `⌥ + X` を押して、電源メニューが出てくるか確認してみましょう

---

## 6. つかってみよう

- **出勤リマインダー**: パソコンを起動 / スリープ解除すると、
  自動でチェックしてお知らせが出ます。何もしなくてOKです。
- **電源メニュー**:
  1. キーボードで `⌥`(option)を押しながら `X` を押す
  2. Power Menu が出てくる
  3. `↑` `↓` でえらんで `Enter`、または `R` `S` `U` `L` を押す
  4. `Shutdown` をえらんだときだけ、退勤かくにん画面が出る
  5. `T` で TeamSpirit を開いて打刻 → `C` でシャットダウン

もういちど `⌥ + X` を押すか `Esc` を押すと、メニューはすぐに閉じます。

---

## 7. TeamSpirit の URL を変えたいとき

`powerMenu.lua` と `teamSpiritReminder.lua`、それぞれのファイルの
いちばん上のほうに、この行があります。

```lua
local TEAMSPIRIT_URL = "https://teamspirit-1227.lightning.force.com/lightning/page/home"
```

会社の TeamSpirit の URL が違う場合は、`" "` の中身だけを
自分の会社のものに書き換えてください。

⚠️ **2つのファイルに、同じ行がそれぞれ1つずつあります。**
片方だけ書き換えると、もう片方は古いURLのままになるので、
必ず **両方のファイル** を書き換えてください。

書き換えたら、忘れずに **「5. 設定を反映させる」** をもう一度やってくださいね。

---

## 8. うまく動かないときは(トラブルシューティング)

| 症状 | 確認すること |
|---|---|
| 出勤のお知らせが出てこない | すでに今日開いた記録が残っているかも。`~/.hammerspoon/teamspirit_last_open.txt` を一度削除してから「Reload Config」してみてください |
| `⌥ + X` を押してもメニューが出ない | Hammerspoon が起動しているか、メニューバーにアイコンがあるか確認してください。アクセシビリティの許可(3-1)も見なおしてみましょう |
| メニューバーに Hammerspoon が出てこない | インストールし直すか、Mac を再起動してみてください |
| エラーが出る | Hammerspoon アイコン →「Console...」を開くと、赤い文字でエラー内容が表示されます。その内容をコピーして質問してください |
| ボタンを押しても TeamSpirit が開かない | インターネットに接続されているか確認してください |
| `require(...)` のところでエラーが出る | `init.lua` の書き方が1文字でも違うと動きません。4-1の手順をもう一度見なおしてください |

---

## 9. このプログラムが記録しているもの

- `~/.hammerspoon/teamspirit_last_open.txt` というファイルに、
  「最後にボタンを押して開いた日付」だけを保存しています
- それ以外の情報(パスワードなど)は一切保存していません

---

## 10. ファイル構成

```
~/.hammerspoon/
├── init.lua                    ← ここから2つのファイルを読み込む
├── powerMenu.lua                ← 電源メニュー本体
├── teamSpiritReminder.lua        ← 出勤リマインダー本体
└── teamspirit_last_open.txt      ← 自動で作られる(さわらなくてOK)
```

そして `⌥ + X` で電源メニューが開くか、ぜひ確認してみてください 👍
